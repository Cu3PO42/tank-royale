Welcome

Welcome to the Robocode docs. These docs are meant for people who are new to Robocode, as well as a reference guide to various aspect of the Robocode Tank Royale programming game.

Get Started

On this page you can pick articles on the menu on the left side to read about various topics.

If you are new to Robocode you should start with the introduction to Robocode, and you should also have a look at My First Bot to learn how to create your first bot.

To run the game and see battles, you should also make sure to install and run the GUI application, unzip the sample bots, set up the bot directories for the sample bots, and start a battle with the sample bots.
